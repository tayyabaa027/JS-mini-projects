# 🚀 JavaScript Mini Projects

A collection of beginner-friendly JavaScript projects built using **HTML**, **CSS**, and **Vanilla JavaScript**. These projects were created while learning core JavaScript concepts such as DOM manipulation, events, timers, asynchronous programming, and game logic.

---

## 📌 Projects Included

| Project | Description |
|---------|-------------|
| 🎨 Background Color Changer | Change the page background color dynamically. |
| ⚖️ BMI Calculator | Calculate Body Mass Index from user input. |
| 🎯 Box Color Changer | Change the color of selected boxes using JavaScript. |
| 🔘 Button Game | Interactive button clicking challenge. |
| 🎮 Color Game | Practice color matching and DOM manipulation. |
| 🌙 Dark Mode Toggle | Switch between Light and Dark themes. |
| 🕒 Digital Clock | Live digital clock using JavaScript timers. |
| 🔢 Guess The Number | Number guessing game with limited attempts. |
| ⌨️ Press Key Display | Display the pressed keyboard key and key code. |
| 🤝 Promise Demo | Basic JavaScript Promise examples. |
| 🐍 Snake Game | Classic Snake game built with Vanilla JavaScript. |
| ❌ Tic Tac Toe | Two-player Tic Tac Toe game. |
| ✅ To-Do List | Add, complete, and remove daily tasks. |
| 🔨 Whack A Mole | Fun reaction-based clicking game. |
| 🌐 XMLHttpRequest Demo | Learn API requests using XMLHttpRequest. |

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/tayyabaa027/JS-mini-projects.git
```

### Navigate to the project folder

```bash
cd JS-mini-projects
```

Open any `.html` file in your browser to run the project.
